21st Century Government

Open Source Projects

City of Chicago belives that open source can help lead to a more efficient, effective, and open government. The city publishes a number of open source projects on GitHub ranging from websites, platforms, research projects, software libraries, and more. By sharing software developed to run Chicago's daily work, the city hopes to provide userful solutions for others. Developers are welcome to use any of the city's software and are invited to report bugs or open pull requests to fix bugs or add new features.
